My Cat V0.5 Playtest

Start:
1. Double-click Install-MyCat.cmd to create desktop and Start Menu shortcuts.
2. If you do not want shortcuts, double-click MyCat.WindowsShell.exe instead.
3. Click the cat to pet it and open its small menu.
4. Use the tray icon menu to record, open settings, inspect folders, or exit.

Try:
- Drag the cat to a desktop spot you want it to stay near.
- Pass the mouse near it and watch for a quiet attention response.
- Tell it the real cat is sleeping, playing, or accompanying you.
- Record the same kind of observation three times in one time period.
- Open behavior rhythm settings and adjust one time period.
- Turn quiet mode on and off from the cat or tray menu.
- Open About My Cat from the tray and confirm the version says V0.5 Playtest.
- Open the data and log folders from the tray.
- Confirm the desktop shortcut, Start Menu shortcut, app exe, and tray all use the My Cat icon.
- Leave quiet mode off long enough to catch an occasional window-side stay.

Notes:
- The cat saves local prototype data and playtest interaction counts under
  %LOCALAPPDATA%\MyCat.
- Logs are saved under %LOCALAPPDATA%\MyCat\logs.
- This is a Windows desktop prototype with the current My Cat runtime art pack.
- The default package is self-contained and should not require a .NET install.
- If something breaks, note what you clicked, whether the tray icon remained,
  whether the app was still visible, and attach the newest log file if possible.
