# My Cat

My Cat 是一只会生活在 Windows 桌面上的小猫。它使用基于真实猫咪绘制的透明美术资产，会坐着、睡觉、慢慢走动、被点击回应、被提起拖动，也会轻轻注意鼠标、窗口边缘和任务栏区域。

它不是效率工具，也不是宠物游戏，而是一个更安静的桌面陪伴原型：你可以把它放在桌面上，让它以低打扰的方式陪你工作、发呆或休息。

## 功能亮点

- 透明无边框桌面小猫窗口，默认置顶显示。
- 自动坐着、睡觉、慢走、伸懒腰和停下观察。
- 点击小猫会有摸摸回应，并短暂看向鼠标。
- 支持拖动小猫，拖动时会呈现被抱起、露出肚皮的动作，松手后自然落下。
- 能感知鼠标、屏幕边缘、窗口边缘和任务栏区域，避免动作太机械。
- 可以记录真实猫咪状态：睡觉、玩、陪我。
- 会根据记录做轻量行为倾向调整。
- 提供“行为节奏设置”，可按早晨、下午、晚上、夜间调整休息、活动、陪伴比例。
- 支持安静模式，减少主动走动和打扰。

## 用户安装

### 方式一：下载发布包

1. 打开 GitHub Releases 页面。
2. 下载最新的 `MyCat-playtest-win-x64.zip`。
3. 解压 zip 到任意文件夹。
4. 双击运行：

```text
MyCat.WindowsShell.exe
```

自包含发布包不需要安装 .NET。启动后，小猫会出现在桌面上，右下角托盘也会出现 My Cat 图标。

### 方式二：从源码运行

如果你是开发者，先安装：

- Windows
- .NET 9 SDK

然后在项目根目录执行：

```powershell
dotnet build .\MyCat.sln --configfile .\NuGet.Config
dotnet run --project .\windows-shell\MyCat.WindowsShell.csproj --no-restore
```

## 使用方式

- 点击小猫：摸摸它，并让它短暂看向鼠标。
- 拖动小猫：按住并移动，松手后它会落下并待在新位置。
- 右键或点击菜单：记录真实猫咪状态、打开行为节奏设置、切换安静模式或退出。
- 托盘图标：也可以记录、设置、切换安静模式和退出。

本地数据保存在：

```text
%LOCALAPPDATA%\MyCat
```

删除这个目录可以重置记录、学习状态、互动计数和行为节奏设置。
