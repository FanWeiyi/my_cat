# Art Pack Image Prompts

The art pack used the built-in image generation path with the five supplied cat
photos as identity references. Transparent sprite-sheet sources were generated
on a flat magenta chroma background, converted to alpha locally, then split
into the final PNG frame directories.

## Identity Basis

```text
Create a polished character identity reference sheet for the same real cat shown
in the five provided reference photos. The cat must remain recognizable as this
exact cat at small desktop-pet scale.

Build one coherent reference sheet with five unclipped studies of the same cat:
front seated view, three-quarter seated view, compact resting or lying view,
close expression study of the face, and tail-marking study. Use a semi-realistic
soft-cute digital illustration style for desktop pet sprite production, with
clean painterly fur masses and simplified fine hairs.

Preserve the rounded short-haired golden-brown shaded tabby identity: round
face, large green eyes, warm pink-brown nose, cream-white chin and muzzle,
lighter cream chest and belly, clear vertical forehead tabby marks, golden eye
surrounds, gray-brown shaded back, dark ringed tail with a blackish tail tip,
and calm companionable temperament.

Use a clean neutral studio reference-sheet background. No accessories, props,
text labels, watermark, extra cats, breed changes, blue eyes, black outlines,
or cropped anatomy.
```

## Idle Sit Sheet

```text
Create one exact 4 column by 4 row production sprite sheet with 16 sequential
frames for the idle_sit action of the locked personalized cat identity.

Use a perfectly flat solid #ff00ff chroma-key background with no floor, shadow,
gradient, texture, or reflection. Put one centered full-body frame in each cell
with a stable ground baseline, stable scale, generous anatomy padding, no cell
borders, and no frame labels.

Animate a calm seated idle loop: settle into still sitting, perform one natural
blink, show a tiny breath and tail-tip movement, then return smoothly to the
opening pose. Preserve the round face, green eyes, pink-brown nose, cream
muzzle and chest, vertical forehead stripes, shaded golden-brown body, ringed
dark tail, and blackish tail tip in every frame.

Use semi-realistic soft-cute digital illustration with simplified painterly fur
masses. No props, hands, accessories, cast shadows, text, watermark, cropped
anatomy, jumpy pose changes, or extra cats.
```

## Expansion Sheets

The remaining action sheets reused this shared prompt shape:

```text
Create one exact 4 column by 4 row production sprite sheet with 16 sequential
frames for the named action of the approved personalized cat identity.

Use a perfectly flat solid #ff00ff chroma-key background with no floor, cast
shadow, reflection, text, label, watermark, prop, accessory, or extra cat.
Keep one full-body cat per cell, stable scale, stable implied ground line, and
generous padding around ears, paws, whiskers, and tail.

Preserve the approved round face, green eyes when visible, pink-brown nose,
cream muzzle and chest, forehead stripes, shaded golden-brown body, ringed dark
tail, blackish tail tip, and semi-realistic soft-cute painterly fur treatment.
```

Action-specific direction:

| Action | Direction |
|---|---|
| `walk_slow_left` | Left-facing calm four-beat slow-walk loop with restrained head bob and tail counterbalance |
| `walk_slow_right` | Right-facing slow-walk loop produced as its own sheet rather than a mirrored afterthought |
| `wake_stretch` | Sleepy rest to low foreleg stretch, then relaxed alert settle with the tail contained inside each cell |
| `edge_stop` | End of a gentle walk, paws gathering into a curious stop and watchful hold |
| `drag_settle` | Soft placement compression, weight shift, blink or tail adjustment, idle-related seated finish |
| `mouse_notice` | Idle attention to ear and eye shift, curious head turn, brief focused hold with no cursor graphic |
| `window_linger` | Quiet sit or loaf watch loop that reads near a window without drawing a window or sill |
| `observation_rest` | Closed-mouth sleepy blink, drowsy lowering, and gentle restful hold |
| `observation_activity` | Perked ears and eyes, contained playful lean, alert return without a toy |
| `observation_accompany` | Warm soft-eyed lean, small nuzzle-like head tilt, calm seated settle |
